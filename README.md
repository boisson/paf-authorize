# ProtesteAuthorize

Authorization using Access Control Application V.2.0

## Installation

Add this line to your application's Gemfile:

    gem 'proteste_authorize', :git => 'git@github.com:proteste/paf-authorize.git'

And then execute:

    $ bundle

## Install

    # rails g proteste_authorize:install
